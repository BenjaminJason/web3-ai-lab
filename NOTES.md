# Web3 · AI · Finance Lab — 设计笔记

> 给小白的"区块链/Web3 × AI × 金融 结合"学习区，分层渐进。
> 本文件是活的设计文档，边讨论边更新。

## 一句话愿景（待打磨）
让完全不懂的人，从零开始、一层层看懂 Web3 + AI + 金融是怎么结合的，重实操、不堆理论。

## 沿用的"招牌打法"（来自前几个项目的经验）
- 单文件 HTML + BYO Key + 静态部署（GitHub Pages / Netlify）
- 三档自适应学习（入门 / 进阶 / 专业），内容随水平加深
- 重实操、反劝退："先让你跑出/看到一个东西"，再讲原理
- 诚实：不吹、标清风险（尤其涉及加密货币/金融）
- 借力开源 + Claude，不自己造轮子

## 已知的最大风险
范围太大 = 什么都想做 = 什么都做不深。必须先切一个**最窄、最有价值**的起点。

## 已定结构（v1）
做一个"学习区骨架"：按【主题分类 × 三档(入门/进阶/专业)】组织，课程卡片填进去。

四大分类（可调）：
- 🌐 Web3 基础：钱包/链/代币/Gas/DeFi/NFT
- 🧠 AI 基础：怎么用好 AI（提示词/Agent/MCP）
- 🔗 AI × 加密：链上数据、项目/币价 AI 解读
- 📊 AI × 金融：财报、投研、量化（复用已有成果）

每个分类下三档：入门 / 进阶 / 专业，每档若干课（lesson 卡片）。
卡片状态：可学 / 即将上线。

三种"借力"全用，按分层组织：
1. Claude 帮生成/策划每节课内容（我审核）
2. 每节课底部嵌"AI 助教"（BYO 免费 Gemini Key，可选）
3. 每个主题附 GitHub 优质开源资源链接

## 执行顺序
1. 先做 hub 骨架（分类×三档 + 卡片 + 路由）+ 1 节完整样板课（Web3 入门）
2. 验证格式 → 再用 Claude 批量填课
3. 部署 GitHub Pages

## 红线（涉及加密/金融必守）
- 不荐币、不荐股、不撮合交易
- 全程"学习/科普"定位 + 免责声明
- 不碰"帮你赚钱"话术
