## ✨ 特点

- **分层学习**：四大分类 × 三档难度（入门 / 进阶 / 专业），13 节课
- **每节都能动手**：不是干读，每课带一个浏览器内沙盒
- **接入真实工具**：财报课跳[实时财报仪表盘](https://benjaminjason.github.io/market-earnings-dashboard/)，量化课跳 [Quant Lab](https://benjaminjason.github.io/quant-lab/)
- **可选 AI 助教**（BYO 免费 Gemini Key）、整合 GitHub 优质开源资源
- **诚实**：涉及加密/投资处一律标注风险，不教「怎么赚钱」

| 分类 | 入门 | 进阶 | 专业 |

## 🛠️ 动手实操举例
生成演示钱包（看清私钥长什么样）· Gas 费模拟器 · DeFi 清算模拟器 · 实时查币价（CoinGecko）· 真实链上地址查询 · 提示词改造器 · 投研提示词生成 · 骗局红旗评分……

## 💻 技术
纯单文件 `index.html`，无后端无构建。Pyodide / ethers.js / Chart.js + 免费数据源。拖到任意静态托管即上线。
